# Skill: configure_hacker_terminal

## Trigger
Use when the user wants an efficient and stylish terminal.

## Safe terminal stack
- zsh
- starship
- fastfetch
- tmux
- fzf
- zoxide
- bat
- eza
- btop
- neovim (optional)

## Style target
- readable prompt
- git status visible
- current directory obvious
- command history searchable
- useful aliases only
- dark aesthetic without harming function

## Do not
- override critical commands unsafely
- hide errors
- add destructive aliases
